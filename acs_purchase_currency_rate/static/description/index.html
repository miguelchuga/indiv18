Currency Exchange Rate in Purchase Order

This module in Odoo allows businesses to handle purchases in foreign currencies with more control over exchange rates. It is especially useful when you deal with international suppliers and want to manage currency rates beyond what the system provides by default.

Module Features

  • Allowes the use of a custom exchabge rate for purchase orders.
  • Computes and stores the total amount in the company's currency using either a custom rate or a system rate.

Form View of Purchase order from That choose currency to apply custom rate.

Make sure choosen currency is not company currency.then and only then you are able to view checkbox.click that checkbox.

Below the subtotal you see the difference according to rate applied.
